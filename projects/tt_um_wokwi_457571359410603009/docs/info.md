## How it works

3-bit binary counter.
Depending on the inputs 0-7, lights up one of the 3 horizontal lines of the 7-segment display
Bottom - 1, Middle - 2, Top - 4

## How to test

If someone wanted to test for 1, when they activate the switch 1, they should see the display light up on the bottom line
Testing for 3 we would see both the bottom and middle lines light up, 5 would be top and bottom and e.c.t
